MERN To-Do App

Folders:
- backend: Node + Express + MongoDB API
- frontend: React app (create-react-app style)

Quick setup:

1) Backend:
   cd backend
   npm install
   # add MONGO_URI in .env
   npm run dev

2) Frontend:
   cd frontend
   npm install
   npm start

Backend default: http://localhost:5000
Frontend default: http://localhost:3000
